
## cpan115

这是一个非官方的115云盘开放平台调用库，可以轻松的在Python中调用115云盘开放平台而不需要多次编写重复的代码

- 下载和上传感觉还是客服端快一些, 这个主用用于linux的操作



### 安装
```bash
pip install cpan115
```

### 使用

- 关于授权参考: [Auth](./auth.md)







### 已实现的接口

- [x] 用户管理
- [x] 文件管理
- [ ] 视频播放
- [ ] 云下载


### 进一步封装的接口.

- [x] 上传文件(夹)
- [x] 下载文件(夹)
- [x] 提供命令行的上传和下载文件(夹)
